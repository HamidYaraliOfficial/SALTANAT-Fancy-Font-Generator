# 🇬🇧 English

## 📌 About The Project

**SALTANAT Fancy Font Generator** is a lightweight web tool that
converts normal text into many creative Unicode font styles.

Features: - ✨ 80+ fancy font styles - 🌍 Persian / English / Chinese
language support - 📱 Responsive design - 📋 One-click copy system - 🎨
Modern glass UI design - ⚡ Pure HTML, CSS, JavaScript project

The project is built as a single HTML application.

------------------------------------------------------------------------

## 🚀 Installation & Run

### Requirements

No backend is required.

You only need: - A modern browser (Chrome, Edge, Firefox) - Internet
connection for external font/icon resources

### Run

1.  Download the project files.
2.  Open:

```{=html}
<!-- -->
```
    fancy_font_generator.html

3.  The application will run in your browser.
